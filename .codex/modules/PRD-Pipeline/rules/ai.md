# Rule 4: AI-Driven Development Notes

This rule covers special considerations for AI-assisted development in CodeTrace-AI.

## Guidelines

- All changes should align with the [AI Development Workflow](docs/Workflow.md)
- Use structured prompts for generating commit messages and PR descriptions
- Ensure specifications are updated alongside code changes
- AI-generated code must follow the project's coding standards
- Document any AI-assisted decisions in commit messages or PR descriptions
