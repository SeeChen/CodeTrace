# Acceptance Contract

Use this reference when drafting acceptance criteria.

## Minimum Sections

1. Purpose
2. Source Documents
3. Milestone Scope
4. Mandatory Gates
5. Required Evidence
6. Deferred or Non-Blocking Checks
7. Open Questions

## Acceptance Standard

Each gate should be:

- specific
- reviewable
- connected to a requirement, constraint, or test plan

The acceptance document should remain concise in size, but each gate should still identify:

- what is being checked
- why it blocks or does not block the milestone
- what evidence is required
