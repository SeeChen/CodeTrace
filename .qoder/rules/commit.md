# Rule 3: Commit Message Guidelines

This rule specifies the format and standards for commit messages in CodeTrace-AI.

## Message Format

- Use imperative mood: "Add feature" not "Added feature"
- Keep first line under 50 characters
- Provide detailed description in commit body if needed
- All commits generated by AI agents must include the signature: "AI-Generated Commit" at the end of the commit message

## Examples

- `feat(workflow): add AI development guidelines`
- `fix(bug): resolve null pointer exception in parser

  Detailed description of the fix and its impact.

  AI-Generated Commit`
